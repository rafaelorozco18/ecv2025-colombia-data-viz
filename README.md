# ecv2025-colombia-data-viz
Data storytelling project on DANE's 2025 Colombia Quality of Life Survey (87K households). Three R choropleth maps (sf, ggplot2, MAGNA-SIRGAS CRS) plus an interactive Power BI dashboard, following the SCQA framework. Reveals an 8x income-insufficiency gap and a 53-point refrigerator-access gap across departments.
